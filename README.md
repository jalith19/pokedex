# Creación de Cuenta en la Nube - Azure for Students

## 1. Requisitos previos
Antes de crear la cuenta en la nube se necesita:

1. Un correo electrónico educativo válido.
2. Verificación del estatus de estudiante.
3. Un número de teléfono para la verificación de la cuenta.

---

## 2. Acceder al portal de Azure for Students
1. Abrir el navegador web.
2. Ir al portal de Azure for Students.
3. Ubicar la opción para crear la cuenta gratuita.

---

## 3. Comenzar el registro
1. Hacer clic en el botón **Empezar gratis**.
2. Esperar a que se cargue la página de inicio de sesión.

---

## 4. Iniciar sesión
1. Introducir el correo electrónico educativo.
2. Hacer clic en **Siguiente**.
3. Introducir la contraseña.
4. Confirmar el inicio de sesión.

---

## 5. Verificar estatus de estudiante
1. Completar el formulario solicitado.
2. Confirmar que se pertenece a una institución educativa.
3. Enviar la información.
4. Esperar la validación del estatus de estudiante.

---

## 6. Verificar la cuenta
1. Introducir un número de teléfono.
2. Recibir el código de verificación por SMS.
3. Ingresar el código recibido.
4. Confirmar la verificación.

---

## 7. Cuenta creada
1. Una vez validada la información, la cuenta queda activa.
2. Se obtiene acceso a Azure for Students.
3. Ya se pueden usar los servicios en la nube.
4. Se puede comenzar a crear recursos en Azure.

---

## 8. Resultado
La cuenta de Azure for Students queda lista para:
- Crear aplicaciones web
- Usar máquinas virtuales
- Implementar bases de datos
- Desplegar proyectos en la nube
